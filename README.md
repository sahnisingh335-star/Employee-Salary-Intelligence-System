
#  Employee Salary Intelligence System

## 📌 Overview
This project is an end-to-end automated salary intelligence system that processes raw employee data, generates KPIs, predicts salaries using machine learning, and produces actionable insights.

## 🚀 Features
- Automated data pipeline (clean → process → analyze → output)
- Data cleaning (missing values, duplicates, standardization)
- KPI engine (average salary, attrition risk, performance metrics)
- Salary prediction using Linear Regression (ML)
- Smart insights generation
- Advanced analysis (salary gap, department efficiency)
- Export system (CSV & JSON reports)

## 📂 Project Structure
- Script/ → Python scripts
- data/ → Input dataset
- Output/ → Generated results
